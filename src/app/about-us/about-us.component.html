Proyecto para demostrar habilidades en Angular, Clean Code, unit testing y accessibility

El Proyecto

Crear una enciclopedia en Angular 16, tratando de implementar lo aprendido en los cursos que tome.

Para la información se usará la api publica de “https://pokeapi.co/docs/v2”.

La app le será posible navegar para personas con discapacidades usando las propiedades “aria” y también contará con un coverage de pruebas unitarias de al menos el 80%.

Menu principal

El menú principal dará la bienvenida al usuario con un mensaje amigable, y mostrará 5 pokemon al azar.

A la izquierda del mismo se mostrará un menú para poder acceder a las distintas secciones.

Seccion: Enciclopedia

Esta sección mostrará un listado con todos los pokemon disponibles. Se mostrará su foto y su  nombre. Arriba habrá un buscador para poder achicar la búsqueda. Al hacer click en el pokemon este se abrirá en pantalla completa con más información detallada

Seccion: Acerca de

Esta sección tendrá información estática sobre el creador del proyecto, e información sobre las tecnologías usadas en su creación.

Tecnologias a usar:

  • Angular 16
  • Javascript
  • Typescript
  • Css
  • Html 5
  • Jasmine
  • Ng-bootstrap (diseño)
  • ngx-infinite-scroll
